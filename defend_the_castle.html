Controls
Desktop
  • Mouse: Place, upgrade, or sell defenses
  • W: Select Wall
  • A: Select Archer Tower
  • C: Select Cannon Tower

  • Undo: Undo last build/upgrade

  • Buttons: Use on-screen UI for actions
